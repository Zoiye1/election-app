## Domain Model

This diagram shows the data structure of the application. There are two main areas:

**Election Data**
Official data about elections, including participating parties, candidates, and the geographical structure (constituencies and municipalities).

**Discussion Platform**
Users can create discussions and reply to each other. By tagging a discussion with an election year, users can filter discussions relevant to a specific election.
